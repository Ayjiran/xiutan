# 幻隐资源嗅探 - Edge浏览器扩展

## 📖 项目介绍

**幻隐资源嗅探**是一款功能强大的Edge浏览器扩展，不仅提供基础的网页资源嗅探功能，还集成了源码查看、可视化编辑、图片预览等高级功能，是开发者、设计师和普通用户的必备神器。

无论您需要下载网页中的图片、音视频资源，还是需要分析网页源代码，甚至想临时修改网页内容进行调试，这款扩展都能满足您的需求。

## ✨ 核心功能

### 🔍 资源嗅探与抓取
- **图片资源**：自动捕获网页中所有图片资源，支持预览、复制链接、新标签打开
- **视频资源**：嗅探网页中的所有视频文件链接，一键获取下载地址
- **音频资源**：提取网页背景音乐、语音音频等音频资源
- **其他资源**：捕获网页中其他类型的资源文件

### 🛠️ 开发者工具
- **源代码查看**：一键获取当前页面完整的HTML源代码
- **源代码复制**：支持一键复制整个页面源代码到剪贴板
- **可视化编辑**：开启可视化编辑模式，像编辑文档一样直接修改网页内容
- **实时资源监控**：动态监听页面资源加载，实时更新资源列表

### 🎨 用户体验
- **玻璃拟态UI**：采用现代化的毛玻璃效果界面设计
- **图片悬停预览**：鼠标悬停在图片链接上即可预览大图
- **一键复制链接**：所有资源链接支持一键复制
- **标签页分类**：资源按类型分类展示，清晰易读
- **实时计数更新**：各分类资源数量实时统计显示

## 🚀 安装方法

### 方法一：开发者模式加载（推荐）

1. **打开Edge浏览器扩展管理页面**
   - 在地址栏输入：`edge://extensions/`
   - 或者点击浏览器右上角三个点 → 扩展 → 管理扩展

2. **开启开发者模式**
   - 在扩展管理页面右上角开启「开发人员模式」

3. **加载解压的扩展**
   - 点击「加载解压缩的扩展」
   - 选择本项目的`edge-extension`文件夹
   - 扩展即可成功安装到浏览器中

### 方法二：打包安装（.crx格式）

如果您想将扩展打包分享给他人，可以：
1. 在扩展管理页面点击「打包扩展」
2. 选择扩展根目录生成.crx文件
3. 他人只需将.crx文件拖入Edge浏览器即可安装

## 🎮 使用指南

### 首次激活
1. 点击浏览器工具栏中的扩展图标打开弹窗
2. 在激活面板输入激活码：**547266**
3. 点击「激活」按钮即可解锁全部功能

### 资源抓取使用
1. 访问任意网页后点击扩展图标
2. 切换到对应的资源标签页（图片/视频/音频/其他）
3. 对每个资源可以：
   - 点击链接在新标签页打开
   - 点击「复制」按钮复制资源链接
   - 点击「预览」按钮在新标签页预览资源
   - 图片资源鼠标悬停即可直接预览大图

### 源代码查看
1. 切换到「源代码」标签页
2. 扩展会自动获取当前页面的完整HTML源代码
3. 点击「复制源代码」按钮可以复制全部代码
4. 点击「开启可视化编辑」可以进入页面编辑模式

### 可视化编辑功能
开启可视化编辑后，您可以：
- 直接点击修改网页上的任何文字内容
- 调整页面元素的位置和样式
- 实时预览修改效果
- 适合做页面演示、调试和临时修改

## ❓ 常见问题

**Q: 扩展无法加载某些网站的资源怎么办？**  
A: 部分网站有特殊的资源保护机制，这是正常现象。本扩展适用于绝大多数普通网站。

**Q: 可视化编辑会修改原网站吗？**  
A: 不会，所有修改仅在您的本地浏览器中生效，刷新页面后就会恢复原状。

**Q: 激活码忘记了怎么办？**  
A: 本扩展的通用激活码是：547266

**Q: 可以在其他Chrome内核浏览器使用吗？**  
A: 可以，本扩展基于Manifest V3开发，兼容所有支持Chrome扩展的浏览器。
